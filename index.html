Live System Monitoring

This website is proudly self-hosted on a Raspberry Pi 5 running the latest version of Debian Linux. It is reverse-proxied through Cloudflare for enhanced global availability, DDoS mitigation, and TLS termination, and actively monitored using Datadog for comprehensive observability.

I use this setup as a practical testbed to explore advanced monitoring strategies, including custom metrics, real-time alerting, and log aggregation. The stack includes Apache2 for serving static content and custom shell scripts for automated backups and patch management.

This environment supports my ongoing learning in infrastructure as code, security hardening, and cloud-native architecture, reflecting my passion for hands-on technical leadership in support engineering.
